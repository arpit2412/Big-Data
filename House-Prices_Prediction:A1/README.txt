
To Run:

Python: python House_Pred.py


Pre-requisites:
CSVs for train and test should be in same folder. 

Libraries required:

1. Numpy
2. Pandas
3. Matplotlib
4. Scipy
5. Seaborn
6. Scikit-learn
7. xgboost
8. lightgbm
9. Maths

For Installing Pip: https://pip.pypa.io/en/stable/installing/
For Installing Conda: https://docs.conda.io/projects/conda/en/latest/user-guide/install/

To Install all the Libraries:

1. Numpy 
For Conda:
	conda install numpy
For Pip:
	pip install numpy

2. Pandas
For Conda:
	conda install pandas
For Pip:
	pip install pandas
3. Matplotlib
For Conda:
	conda install -c conda-forge matplotlib 
For Pip:
	pip install -U matplotlib
4. Scipy
For Conda:
	conda install -c anaconda scipy
For Pip:
	pip install scipy
5. Seaborn
For Conda:
	conda install seaborn
For Pip:
	pip install seaborn
6. Scikit-learn
For Conda:
	conda install scikit-learn 
For Pip:
	pip install -U scikit-learn
7. XGBoost
For Conda:
	conda install -c anaconda py-xgboost
For Pip:
	pip install xgboost
8. LightGBM
For Conda:
	conda install -c conda-forge lightgbm
For Pip:
	pip install lightgbm>
9. Maths
For Conda:
	conda install -c auto mathematics
For Pip: 
	pip install maths

To Run:

Python: python House_Pred.py
